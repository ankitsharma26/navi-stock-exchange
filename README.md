## Run Application

Go to ./target RUN

```
java -jar stock-0.0.1-SNAPSHOT.jar ../file/test.txt
```

## For Custom File

Go to ./target RUN

```
java -jar stock-0.0.1-SNAPSHOT.jar <file-path>
```

## MVN Build

GO TO ROOT REPOSITORY FOLDER RUN

```
mvn clean install
```
